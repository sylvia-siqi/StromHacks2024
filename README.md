# StormHacks2024 - FitCat
Reach your fitness and health goals and raise your own cat! <br/>
 Tech stack:
 React, Flask, SQLite <br/><br/>


When installing: (installs all of the packages you need) <br/>
cd backend <br/>
npm i <br/>
cd .. <br/>
cd frontend <br/>
npm i <br/><br/>

To run code: <br/>
cd backend <br/>
flask run <br/>
(add new terminal) <br/>
cd frontend <br/>
npm start
